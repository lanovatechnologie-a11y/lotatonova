# Nova Lotto - Système de Gestion de Loterie

## Déploiement
1. Connectez ce dépôt à Render.com
2. Configurez Supabase avec le script SQL
3. Accédez à l'URL Render fournie

## Accès
- Agent: `/index.html` → `lotato.html`
- Superviseur: `/control-level1.html` et `/control-level2.html`
- Administrateur: `/subsystem-admin.html`
- Master: `/master-dashboard.html`
